Sloth’s Ledger App

Sloth’s Ledger is a manual personal finance app built for clarity, privacy, and control.

Download

A pre-release version of Sloth's Ledger is available for free from the link below.
Alternatively, you can clone the GitHub repository and build it yourself, or download a release from the GitHub releases page.

Download Pre-Release Version

How It Works

Accounts are created manually and transactions are entered by the user. There is no bank connection and no automatic syncing. Every balance exists because it was recorded.

Transactions are grouped by day with a running total, making it easy to understand how money moves.

Subscriptions

A built-in subscription tracker shows upcoming payments, highlights overdue items, and can automatically create a ledger entry when marked as paid.

Philosophy

This project is developed independently and evolves over time. It is designed to stay simple and intentional.
